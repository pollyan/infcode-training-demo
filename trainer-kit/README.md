# trainer-kit

这是讲师专用入口目录。

如果你要带客户做演练，建议从这里开始，不需要先翻整个项目。

## 建议阅读顺序

1. `SETUP.md`
2. `TRAINER_RUNBOOK.md`
3. `教学对照清单.md`

## 每份文档怎么用

### `SETUP.md`

用于开课前准备环境。

你可以在这里看到：

- 启动顺序
- 启动命令
- 分支使用建议
- 培训时推荐引用哪些材料

### `TRAINER_RUNBOOK.md`

用于讲师现场控节奏。

你可以在这里看到：

- 每个阶段该怎么带
- 现场该强调什么
- 学员跑偏时如何纠偏
- 卡住时如何兜底

### `教学对照清单.md`

用于讲师备课和课后复盘。

你可以在这里看到：

- `training/base` 和 `training/done` 的差异
- 学员需要补哪些内容
- 讲师应该重点看哪些点

## 讲师工作流建议

### 课前

1. 先看 `SETUP.md`
2. 在本地跑一遍环境
3. 切到 `training/done` 熟悉参考实现

### 上课时

1. 切回 `training/base`
2. 按 `TRAINER_RUNBOOK.md` 的节奏带练
3. 必要时用 `教学对照清单.md` 做快速对照

### 课后

1. 用 `教学对照清单.md` 做复盘
2. 记录学员最容易卡住的环节
3. 必要时补充新的故障场景和提示词
