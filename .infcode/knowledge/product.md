# 产品概述

## 产品概览

InfCode 基础练习项目是一个用于培训和演示的客户信息管理系统。该项目采用前后端分离架构，通过集成外部客户中心服务，为用户提供客户信息查询和同步功能。项目主要用于教学场景，帮助学员理解和实践前后端开发、外部服务集成以及分层架构设计。

## 核心功能

### 1. 客户目录查询
- **功能描述**：用户通过前端页面查询客户目录列表，支持按关键词、客户状态、同步状态进行筛选
- **查询内容**：客户编码、客户名称、客户状态、同步状态、联系电话、更新时间等
- **筛选能力**：支持多维度组合筛选，提供灵活的查询体验

### 2. 客户资料同步
- **功能描述**：从外部客户中心服务同步最新的客户资料信息
- **同步方式**：用户点击同步按钮，系统调用外部 API 获取最新数据并更新本地存储
- **状态管理**：同步后更新客户的同步状态和更新时间

### 3. 外部服务集成
- **集成对象**：客户中心外部服务（通过 Mock Server 模拟）
- **集成方式**：后端通过 HTTP 客户端调用外部 API，获取客户数据
- **数据映射**：将外部服务返回的字段映射为内部 DTO 对象

### 4. 健康检查
- **功能描述**：提供后端服务健康状态检查接口
- **用途**：用于监控服务运行状态，确保系统可用性

### 5. 前端单页应用
- **功能描述**：基于 Hash 路由的单页应用
- **交互方式**：提供客户目录页面，支持查询、筛选和同步操作

## 培训价值

该项目作为培训工具，帮助学员掌握：
- 前后端分离架构的实现方式
- RESTful API 的设计与调用
- 外部服务集成的最佳实践
- 分层架构（Controller、Service、Integration）的应用
- 错误处理和用户体验优化
